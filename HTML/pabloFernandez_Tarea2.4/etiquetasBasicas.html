Etiquetas Básicas

Etiquetas Básicas de Párrafo

  • Parrafo Simple <p>: indica un elemento de párrafo normal.
  • Encabezados <h1> <h2> <h3> <h4> <h5> <h6>: Representan elementos de encabezado, dotando a cada uno de ellos de distinto nivel de importancia.
  • Salto de Linea <br>: Salta a la siguiente linea dentro de un párrafo.
  • Linea Horizontal <hr>: Genera una linea y se suele situar entre las etiquetas de párrafo.

Etiquetas de Marcado Simple

  • Texto importante <strong>: Se representa en el navegador poniendo las letras de dentro de su etiqueta en negrita.
  • Texto enfatizado <em>: Se represta en el navegador poniendo las letras de dentro de su etiqueda en cursiva.
  • Texto resaltado <mark>: Remerca el texto dentro de su etiqueta en color amarillo.

Citas

  • Cita literal <q>: Indicar citas cortas incluidas en un párrafo con otro tipo de texto. Y su atributo cite contiene la URL de la que se ha extraido la cita.
  • Referencia <cite>: se recomienda su uso para indicar títulos de trabajo y el nombre de sus autores.
  • Cita de Parrafo <blockquote>: Esta etiqueta se utiliza para marcar que parrafos completos son citas procedentres de fuentes externas al documento. Y su atributo cite contiene la URL de la que se ha extraido la cita.

Otros elementos de marcado de caracteres

  • Subíndices y superíndices <sub> <sup>:
    • <sub>: El texto aparece por debajo de la linea base y en un tamaño más pequeño.
    • <sup>: El texto aparece por encima de la linea base y en un tamaño más pequeño.
  • Tachado <s>: Muestra el texto indicado tachado con una linea horizontal.
  • Subrayado <s>: Muestra el texto indicado subrayado con una linea horizontal.
  • Negrita <b>: Muestra el texto indicado en negrita.
  • Cursiva <i>: Muestra el texto delimitado en cursiva.
  • Abreviatura <abbr>: Indica una abreviatura o un acrónimo y requiere del atributo title para indicar el significado de la abreviatura.
  • Definiciones <dfn>: Sirve para indicar definiciones para el término y requiere el atributo title para indicar la definicion.
  • Escritura de código <code>: Se utiliza cuando en un documento se desea escribir ejemplos de codigo usadon un lenguaje de programacion.
  • Resultados de operaciones de un ordenador <samp>: Se utiliza cuando el texto pertenece a un mensaje perteneciente a un ordenador.
  • Teclas <kbd>: Sirve para indicar el nombre de una tecla o combinacion de estas.
  • Variables <var>: Sirve para indicar el nombre de una variable.
  • Fechas y horas <time>: Sirve para indicar fechas y horas.
  • Direccion del texto <bdi> <bdo>: Sirve para indicar el sentido del texto, de izquierda a derecha o de derecha a izquierda.
  • Division de palabras <wbr>: Sirve para indicar la division de palabras.
  • Delimitador de formato <span>: Sirve para marcar un trozo de texto de forma insepecífica, su uso habitual es para el css.
  • Direccion <address>: Sirve para indicar la direccion de una persona.
  • Marcado de Correcciones <del> <ins>: Sirve para indicar correcciones.

Elementos para acciones interativas

  • Elementos <details> <summary>:
    • <details>: Sirve para crear un contenedor que se expande o se contrae con el uso de un boton.
    • <summary>: Sirve para definir un resumen o leyenda para el elemento <details>.
  • Elemento <dialog>: Sirve para crear un contenedor que se expande o se contrae con el uso de un boton.

Listas

  • Listas no ordenadas <ul> <li>: Sirve para crear listas de elementos sin un orden específico.
    • <ul>: Sirve para crear una lista de elementos sin orden.
    • <li>: Sirve para crear un elemento de la lista.
  • Listas ordenadas <ol> <li>: Sirve para crear listas de elementos con un orden específico.
    • <ol>: Sirve para crear una lista de elementos con orden.
    • <li>: Sirve para crear un elemento de la lista.
  • Listas de definición <dl> <dt> <dd>: Sirve para crear listas de definiciones.
    • <dl>: Sirve para crear una lista de definiciones.
    • <dt>: Sirve para crear un término de la definición.
    • <dd>: Sirve para crear la descripción del término de la definición.
