Etiquetas y elementos

Las etiquetas son las instrucciones que se utilizan para definir los diferentes elementos de un documento HTML. Estas etiquetas se escriben entre corchetes angulares (< >) y suelen venir en pares: una etiqueta de apertura y una etiqueta de cierre. Por ejemplo, la etiqueta <p> se utiliza para definir un párrafo, y su etiqueta de cierre es </p>.

<etiqueta> Esto es el contenido de la etiqueta </etiqueta>

Atributos

Los atributos proporcionan información adicional sobre un elemento HTML. Se colocan dentro de la etiqueta de apertura y suelen tener un valor asignado. Por ejemplo, el atributo "href" en la etiqueta <a> se utiliza para especificar la URL a la que se enlaza.

<etiqueta atributo1="valor1" atributo2="valor2">

Comentarios

Los comentarios en HTML se utilizan para agregar notas o explicaciones dentro del código que no serán visibles en el navegador. Los comentarios comienzan con <!-- y terminan con -->.

<!-- Este es un comentario en HTML -->

Codigos de idioma

El código de idioma se especifica en el elemento <html> ; mediante el atributo lang. Este atributo indica el idioma principal del contenido del documento. Por ejemplo, para indicar que el contenido está en español, se utiliza el código "es".

Estructura de un documento HTML

<DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title></title> </head> <body> </body> </html>

Elemento <head>

Contiene el titulo de la pagina con <title> y los metadatos <meta> que tienen efecto sobre la representacion de la web en el navegador

Tipos de metadatos

  1. charset="": Define la codificación de caracteres del documento.
  2. http-equiv + content
    • http-equiv="Content_Type" content="text/html; charset=UTF-8": Especificar el tipo de contenido y el juego de caracteres
    • http-equiv="cache-control" content="no-store": Especificar cómo debe gestionar el navegador desde el punto de vista de la caché>
    • http-equiv="refresh" content="30": Indicar al navegador que refresque la página cada 30 segundos.
  3. name + content
    • name="keywords" content="LLMM, marcas, html": Indicar palabras clave, relacionadas con el contenido, para que los indexadores de los motores de búsqueda puedan clasificar la página de manera más eficiente.
    • name="author/copyright" content="El nombre del autor/nombre del copyright": Indicar el nombre del autor o el propietario de los derechos de la página web
    • name="viewport" content="width=device-width, initial-scale=1.0": Determinar el ancho de la zona de representación (viewport) de la página,

Elemento <body>

Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, tablas, formularios, etc. Es la parte principal del documento HTML y es donde se define la estructura y el diseño de la página.

Estructura de una pagina
  • <header>: Define el encabezado de la página, que generalmente contiene el logotipo, el título y la navegación principal.
  • <nav>: Define una sección de navegación que contiene enlaces a otras partes del sitio web.
  • <main>: Define el contenido principal de la página, que es único para cada página y representa la parte más importante del contenido.
  • <section>: Define una sección temática dentro del contenido principal, que puede contener varios elementos relacionados.
  • <article>: Define un artículo independiente dentro del contenido, como una entrada de blog o una noticia.
  • <aside>: Define contenido adicional o complementario que está relacionado con el contenido principal, como barras laterales o anuncios.
  • <footer>: Define el pie de página de la página, que generalmente contiene información de contacto, derechos de autor y enlaces adicionales.
Atributos globales

Los atributos globales son aquellos que pueden aplicarse a cualquier elemento HTML. Algunos de los atributos globales más comunes son:

  • id: Asigna un identificador único a un elemento.
  • class: Asigna una o más clases a un elemento para aplicar estilos CSS.
  • style: Permite aplicar estilos CSS en línea a un elemento.
  • title: Proporciona información adicional sobre un elemento, que se muestra como un tooltip al pasar el cursor sobre él.
  • contenteditable: Si existe, el contenido del elemento es editable
  • hidden: Si existe, el elemento no se representa en el navegador
  • lang: Determina el idioma en el que esta escrito el contenido del elemento
  • spellcheck: Determina si el elemento debe ser analizado ortograficamente y gramaticalmente por el navegador
  • tabindex: Determina el orden de seleccion de los elementos cuando se pulsa Tab
  • Translate: Determina si el elemento debe ser traducido o no
