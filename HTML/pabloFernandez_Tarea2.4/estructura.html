Estructura

Los elementos estructurales son aquellos que permiten la estructura de la pagina web

Elemento <div>

El elemento <div> es un contenedor genérico que se utiliza para agrupar otros elementos HTML y aplicarles estilos o scripts de manera conjunta. No tiene un significado semántico específico, pero es muy útil para organizar el contenido y crear diseños complejos en una página web.

<div class="contenedor"> <h2>Título dentro del div</h2> <p>Párrafo dentro del div.</p> </div>

Elemento <span>

El elemento <span> es un contenedor en línea que se utiliza para aplicar estilos o scripts a una parte específica del texto dentro de un elemento más grande, como un párrafo o un encabezado. Al igual que <div>, <span> no tiene un significado semántico específico, pero es útil para resaltar o modificar partes del contenido sin afectar la estructura general del documento.

<p>Este es un párrafo con un <span style="color: red;">texto resaltado en rojo</span> dentro del párrafo.</p>

Elemento <header>

El elemento <header> es un contenedor que se utiliza para definir el encabezado de una página web. Contiene el logotipo, el título de la página y la navegación principal.


                <header>
                    <img src="logo.png" alt="Logotipo de la página">
                    <h1>Título de la página</h1>
                    <nav>
                        <ul>
                            <li><a href="#inicio">Inicio</a></li>
                            <li><a href="#sobre-nosotros">Sobre Nosotros</a></li>
                            <li><a href="#contacto">Contacto</a></li>
                        </ul>
                    </nav>
                </header>

Elemento <footer>

El elemento <footer> es un contenedor que se utiliza para definir el pie de página de una página web. Contiene información de contacto, derechos de autor y enlaces adicionales.

<footer> <p>© 2024 Mi Sitio Web. Todos los derechos reservados.</p> <p>Contacto: <a href="mailto:

Elemento <main>

El elemento <main> es un contenedor que se utiliza para definir el contenido principal de una página web. Representa la parte más importante del contenido y es único para cada página.

<main> <h2>Contenido Principal</h2> <p>Este es el contenido principal de la página.</p> </main>

Elemento <section>

El elemento <section> es un contenedor que se utiliza para definir una sección temática dentro del contenido principal. Puede contener varios elementos relacionados.

<section> <h2>Sección Temática</h2> <p>Este es el contenido de la sección temática.</p> </section>

Elemento <article>

El elemento <article> es un contenedor que se utiliza para definir un artículo independiente dentro del contenido. Puede ser una entrada de blog, una noticia o cualquier otro contenido autónomo.

<article> <h2>Artículo Independiente</h2> <p>Este es el contenido del artículo independiente.</p> </article>

Elemento <aside>

El elemento <aside> es un contenedor que se utiliza para definir contenido adicional o complementario relacionado con el contenido principal. Puede incluir barras laterales, anuncios o información relacionada.

<aside> <h2>Contenido Complementario</h2> <p>Este es el contenido adicional relacionado con el contenido principal.</p> </aside>

Elemento <nav>

El elemento <nav> es un contenedor que se utiliza para definir una sección de navegación en una página web. Contiene enlaces a otras partes del sitio web o a diferentes secciones dentro de la misma página.

<nav> <ul> <li><a href="#inicio">Inicio</a></li> <li><a href="#sobre-nosotros">Sobre Nosotros</a></li> <li><a href="#contacto">Contacto</a></li> </ul> </nav>

Elementos <figure> y <figcaption>

El elemento <figure> se utiliza para agrupar contenido multimedia, como imágenes, diagramas o ilustraciones, junto con su leyenda. La leyenda se define utilizando el elemento <figcaption>.

<figure> <img src="imagen.jpg" alt="Descripción de la imagen"> <figcaption>Esta es la leyenda de la imagen.</figcaption> </figure>
