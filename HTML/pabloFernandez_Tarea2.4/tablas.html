Tablas

Estructura Básica: <table>, <tr> y <td>

La estructura fundamental de una tabla se basa en tres elementos:

  • <table>: Delimita toda la tabla y es el contenedor principal.
  • <tr>: Representa una fila de la tabla (Table Row). Contiene las celdas de la fila.
  • <td>: Representa una celda de datos (Table Data) dentro del cuerpo de la tabla.

Elementos de Marcado Semántico

HTML cuenta con elementos específicos para dar significado estructural a las diferentes partes de la tabla:

  • <caption>: Proporciona un título descriptivo a la tabla. Debe ser el primer elemento dentro de <table>.
  • <thead>: Contiene el conjunto de filas que forman el encabezado de la tabla. Suele ir antes del cuerpo.
  • <tbody>: Delimita el cuerpo de la tabla, que contiene los datos principales. Una tabla puede tener varios <tbody>.
  • <tfoot>: Contiene el conjunto de filas que forman el pie de la tabla. Se usa para mostrar totales, acumulados o resúmenes.
  • <th>: Representa una celda de encabezado. Su contenido aparece en negrita por defecto y se utiliza dentro de <thead> o <tfoot>.

Agrupación de Columnas y Combinación de Celdas

Agrupación de Columnas: <colgroup> y <col>

Estos elementos se usan para agrupar columnas y aplicarles un formato homogéneo o propiedades:

  • <code><colgroup>: Permite realizar agrupaciones de columnas. Debe ser de las primeras etiquetas dentro de <table>.
  • <col>: Se usa dentro de <colgroup> para aplicar propiedades a las columnas. El atributo span determina el número de columnas consecutivas a agrupar.

Combinar Celdas

Se pueden unir celdas horizontal o verticalmente usando atributos dentro de los elementos <td> o <th>:

  • colspan: Determina el número de celdas consecutivas hacia la derecha que debe agrupar. (Ej: colspan="3" une tres celdas en horizontal).
  • rowspan: Determina el número de celdas consecutivas hacia abajo que debe agrupar. (Ej: rowspan="3" une tres celdas en vertical).
