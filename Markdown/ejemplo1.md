# IMPORTANTE
Los navegadores no abren arcjivos markdown de forma nativa (lo muestran como un archivo de texto)
- Uso de extensiones (Markdown Viewer)
- En editores de código como VS Code basta con abrir la vista previa en tiempo real (`Ctrl + Shift + V`)
- Plataformas como GitHub renderizan automaticamente estos archivos para que se vean como páginas web (README.md)
