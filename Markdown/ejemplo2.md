# Guía de Instalación: Entorno de Desarrollo Java

Esta guía describe los pasos necesarios para configurar tu equipo para el módulo de Programación.

## 1. Requisitos Previos
* Sistema Operativo: Windows 10 o superior / macOS / Linux.
* Espacio en disco: Mínimo 2GB libres.
* Conexión a Internet.

---

## 2. Instalación del JDK (Java Development Kit)
El JDK es el software necesario para compilar y ejecutar programas en Java.

1. Descarga el instalador desde la web oficial de [Oracle](https://www.oracle.com/java/).
2. Ejecuta el instalador siguiendo los pasos predeterminados.
3. **Verificación**: Abre una terminal (_cmd_ o _PowerShell_) y escribe el siguiente comando:

```bash
java -version
```

**Importante:** Si la terminal dice que el comando no se reconoce, debes revisar las variables de entorno del sistema.
